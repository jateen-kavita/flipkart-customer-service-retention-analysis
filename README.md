# 📞 Flipkart — Customer Service Performance & Retention Analysis

## 📌 Project Overview
This repository contains a **SQL-based analytics case study** analyzing
Flipkart’s customer service performance and its impact on customer retention.

The analysis focuses on support ticket trends, resolution efficiency, agent
performance, and repeat customer behavior to identify opportunities for
improving customer experience and loyalty.

## 📊 What’s Included
- Raw and cleaned datasets related to customer support
- SQL queries answering business-driven questions
- Consolidated case study with query outputs and insights
- Stakeholder-ready presentation summarizing findings

## 🛠 Tools Used
- CSV datasets
- Excel (data validation)
- Presentation (PDF)

## 📁 Repository Structure
- `data/` → support and customer datasets
- `excel analysis/` → performance and retention analysis queries
- `presentation/` → final insights deck
